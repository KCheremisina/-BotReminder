# Telegram Bot для управления циклическими событиями

Телеграм бот для автоматической отправки уведомлений о циклических событиях с поддержкой прокси.

## Возможности

- 🔄 Циклическое расписание событий (47-дневный цикл)
- 📅 Автоматическое включение/отключение уведомлений о событиях
- ⏰ Планирование ежедневных сообщений
- 🔐 Поддержка прокси (SOCKS4, SOCKS5, HTTP, HTTPS) - в разработке
- 📊 Команды для просмотра статуса и информации о событиях
- 💾 Автоматическое создание бэкапов конфигурации

## Установка

### Требования

- Python 3.8 или выше
- pip (менеджер пакетов Python)

### Шаги установки

1. Клонируйте репозиторий:

git clone <repository-url>
cd <project-directory>

2. Установите зависимости:
pip install -r requirements.txt

##  Команды Бота
/start	Показать информацию о боте и расписании
/status	Показать статус всех сообщений и активных событий
/gimmeinfo	Показать информацию о сегодняшних событиях
/events [YYYY-MM-DD]	Показать события на указанную дату
/reload	Перезагрузить конфигурацию без перезапуска бота

##  Структура проекта
project/
├── bot.py                 # Основной файл бота
├── cicle.py              # Класс для управления циклическими событиями
├── proxy_manager.py      # Менеджер прокси
├── config.py             # Конфигурация (создается пользователем)
├── messages_config.json  # Конфигурация сообщений (опционально)
├── requirements.txt      # Зависимости
└── README.md            # Документация

## Примечание по MTProto
MTProto прокси не поддерживаются в данной версии. Эта функция еще в разработке.


## Автоматическое обновление
Бот автоматически обновляет конфигурацию сообщений каждый день в 00:00:10
    Включает уведомления для активных событий
    Отключает уведомления для неактивных событий
    Создает бэкапы конфигурации перед изменениями
